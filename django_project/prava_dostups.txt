Правила доступа:
Администратор (admin):
Может создавать, редактировать, удалять, просматривать, менять статус активности любых питомцев.
Может видеть всех питомцев (активных и неактивных).
Модератор (moderator):
Может просматривать всех питомцев (активных и неактивных).
Может менять статус активности.
Не может создавать, редактировать или удалять питомцев.
Пользователь (user):
Может создавать, редактировать, удалять только своих питомцев.
Может видеть только активных питомцев (включая свои).
Анонимный пользователь:
Может видеть только активных питомцев.
